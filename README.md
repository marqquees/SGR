# SGR - Sistema Gestor de Reparações

![Badge em Desenvolvimento](http://img.shields.io/static/v1?label=STATUS&message=EM%20DESENVOLVIMENTO&color=GREEN&style=for-the-badge)

## 📋 Sobre o Projeto

O **SGR** é uma aplicação web desenvolvida em **.NET (C#)** e **Blazor**, projetada para otimizar processos de gestão e controle de equipamentos eletrônicos por reparar. O sistema foca em oferecer uma interface intuitiva e funcionalidades essenciais para a administração eficiente de dados.

## 🚀 Funcionalidades Principais

* **Gestão de Cadastros (CRUD):** Criação, leitura, atualização e exclusão dos dados.
* **Ciclo de Vida:** Ciclo de vida dos registos perante os processos/procedimentos.
* **Geração de Relatórios:** Geração de relatórios em PDF utilizando a biblioteca QuestPDF.

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **C# 14**
* **Blazor**
* **NET 10**
* **Entity Framework Core**
* **QuestPDF**
* **HTML5 & CSS3**
* **Bootstrap**
* **SQLite**
